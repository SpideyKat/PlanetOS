# PlanetOS

A website that can potentially save the world.

## Description

This website explains to users the problem that the world is facing right now: Climate change. It includes immersive scroll‑based environments that educate users on environmental issues. It also provides an interactive Carbon Footprint calculator for users, acting as a solution to the problem.  

## Getting Started

### Dependencies

* Any modern web browser (Chrome, Firefox, Safari, Edge)
* No external libraries required — uses pure HTML, CSS, and JavaScript
* Optional: A local server (VSCode Live Server or similar) for smoother testing
